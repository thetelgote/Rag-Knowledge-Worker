# Rag-Knowledge-Worker
A Retrieval-Augmented Generation (RAG) system using LangChain, FAISS, and Ollama. Supports PDF, DOCX, TXT, and Markdown files with a Gradio-based chat interface for intelligent document querying.


## 🔄 Project Updates

### ✅ Version 1 - Basic RAG
- Supports Markdown files
- FAISS indexing

### ✅ Version 2 - Multi-file Support
- Added PDF, DOCX support
- Improved ingestion pipeline

### ✅ Version 3 - Upload Feature
- Users can upload files
- Auto re-indexing added

### ✅ Version 4 - Improved Retrieval
- Fixed incorrect document matching
- Added filename-based filtering
